支援範圍

公開線上版

  • Word task pane 載入
  • Markdown 匯入與匯出
  • manifest 分發與 GitHub Pages 靜態資源

不在公開支援範圍內

  • Windows .md 檔案關聯
  • registry 覆寫
  • 本機 localhost launcher bridge

回報問題前請先確認

  1. 你使用的 manifest URL 是否是 https://github.colorgeek.co/word-markdown-addin/manifest.store.xml。
  2. Word 內 task pane 的來源是否是 https://github.colorgeek.co/word-markdown-addin/taskpane.html，而不是 localhost。
  3. 是否可用 repo 內的 samples/official-smoke-sample.md 重現問題。
  4. 請記錄 Word Desktop 或 Word Online、版本、以及錯誤畫面或訊息。
